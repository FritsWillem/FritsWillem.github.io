Windows command prompt

DCOMCNFG get DCOM configuration
hostname get computer name
MMC open Microsoft Management Console (managing certificates, etc.)
net statistics workstation startup time computer
netstat -abn which ports are in use by which processes
net group [groupName] /domain find Windows users for domain group
net user [userName] /domain find Windows domain groups for user
nslookup [ip-adres] check DNS-server
ping [servername/ ip address] -t continuous ping
shutdown -r -t [number of seconds] reboot pc after x seconds
telnet [server/machine/device] [port] check of specific port can be reached
whoami /groups check which groups the current user is in
